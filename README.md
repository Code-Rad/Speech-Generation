# VoiceForge — AI News Speech Generation Platform

VoiceForge is a modular, broadcast-quality text-to-speech platform built as the speech
generation module of a larger AI News Video Generation Pipeline. It accepts a text script
as input and outputs professional-grade voice audio in English, Hindi, or Hinglish using
a swappable engine system (Edge TTS, XTTS v2, Voxtral TTS, Fish Speech S2 Pro) — all
free and open-source, routed automatically based on language and GPU availability.
